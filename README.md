# 인터랙티브 사업계획서 — 닥프렌즈(Docfriends)

닥프렌즈의 사업 현황, 제품 전략, 재무 정보를 기반으로 인터랙티브 사업계획서를 구성하기 위한 리소스 저장소입니다.

## 프로젝트 구조

```
interactive-proposals/
├── docs/                           # 사업계획서 및 발표 문서
│   ├── docfriends-business-overview.md   # 사업·제품 현황 정리 (영문)
│   ├── 닥프렌즈 피치덱_*.pdf              # 투자용 피치덱
│   ├── 닥프렌즈 소개서(미팅용)_*.pdf       # 미팅용 회사 소개서
│   ├── 닥톡AI 의료광고 자동심의(PT)_*.pdf  # 대한의사협회 발표 자료
│   └── skills.md                         # PRD 생성 스킬 정의
├── resources/
│   ├── images/                     # 서비스별 제품 스크린샷·소개 이미지
│   │   ├── 닥톡 예약 서비스/             # 닥톡 예약 서비스 소개 이미지
│   │   ├── 닥톡 전문가QnA 서비스/        # 전문가 QnA 서비스 소개 이미지
│   │   └── 닥프렌즈 소개/               # 회사 소개 이미지
│   └── references/                 # 전략 참고 자료
│       ├── 2026 Docfriends Business·Product KPI.png
│       └── 2026 Docfriends Product Flywheel.jpg
├── financials/                     # 재무 자료
│   └── [닥프렌즈] 2025년 연결재무제표_*.pdf
└── metrics/                        # 성과 지표 (예정)
```

## 폴더 설명

| 폴더 | 용도 |
|------|------|
| `docs/` | 사업 개요, 피치덱, 소개서, 발표 자료, PRD 스킬 정의 |
| `resources/images/` | 닥톡 예약·전문가QnA·회사 소개 등 서비스별 제품 이미지 |
| `resources/references/` | 2026년 KPI 목표 및 Product Flywheel 전략 다이어그램 |
| `financials/` | 연결재무제표 등 재무 관련 문서 |
| `metrics/` | 사업 성과 지표 (추후 추가 예정) |

## 닥프렌즈 사업 요약

- **비전**: "나만의 주치의, 모든 환자에게 맞춤 의료를"
- **핵심 서비스**: 닥톡 예약 (32+ EMR 연동, 78,000+ 의료기관 시장 점유율 51.3%)
- **마케팅 레이어**: 닥톡 전문가 QnA (네이버 지식iN 연계 의료 전문 답변)
- **개발 중**: 닥톡 AI Agent (AI 상담), 닥톡 Ads (성과 기반 광고 중개)
- **재무 상태**: Default Alive, 35개월 런웨이 확보

자세한 내용은 [`docs/docfriends-business-overview.md`](docs/docfriends-business-overview.md) 참고.
