# 인터랙티브 사업계획서 — 닥프렌즈(Docfriends)

닥프렌즈의 사업 현황, 제품 전략, 재무 정보를 기반으로 인터랙티브 사업계획서를 구성하기 위한 리소스 저장소입니다.

## 프로젝트 구조

```
interactive-proposals/
├── business/                       # 사업 및 서비스 문서
│   ├── docfriends-business-overview.md       # 사업·제품 현황 정리 (영문)
│   ├── doctalk-medical-appointment.md        # 닥톡 예약 서비스 개요
│   ├── doctalkai-autoad-reviewer.md          # 닥톡 AI 의료광고 자동심의 개요
│   └── doctalk-ad(DSP).md                    # 닥톡 Ads DSP/DMP 전략
├── resources/
│   ├── images/                     # 서비스별 제품 스크린샷·소개 이미지
│   │   ├── 닥톡 예약 서비스/             # 닥톡 예약 서비스 소개 이미지
│   │   ├── 닥톡 전문가QnA 서비스/        # 전문가 QnA 서비스 소개 이미지
│   │   └── 닥프렌즈 소개/               # 회사 소개 이미지
│   └── references/                 # 전략 참고 자료
│       ├── 2026 Docfriends Business·Product KPI.png
│       ├── 2026 Docfriends Product Flywheel.jpg
│       └── 닥톡AI 의료광고 자동심의(PT)_*.pdf
├── financials/                     # 재무 자료
│   └── [닥프렌즈] 2025년 연결재무제표_*.pdf
├── metrics and KPIs/               # 성과 지표 (예정)
└── skills/                         # 스킬 정의
    └── PRD-generator.md                # PRD 생성 스킬
```

## 폴더 설명

| 폴더 | 용도 |
|------|------|
| `business/` | 사업 개요, 서비스별 상세 문서 (예약, AI 심의, 광고 DSP) |
| `resources/images/` | 닥톡 예약·전문가QnA·회사 소개 등 서비스별 제품 이미지 |
| `resources/references/` | KPI 목표, Product Flywheel 전략, 발표 자료 PDF |
| `financials/` | 연결재무제표 등 재무 관련 문서 |
| `metrics and KPIs/` | 사업 성과 지표 (추후 추가 예정) |
| `skills/` | PRD 생성 등 작업 스킬 정의 |

## 닥프렌즈 사업 요약

- **비전**: "나만의 주치의, 모든 환자에게 맞춤 의료를"
- **핵심 서비스**: 닥톡 예약 (32+ EMR 연동, 25,000 클리닉, 시장 점유율 32%)
- **마케팅 레이어**: 닥톡 전문가 QnA (네이버 지식iN 연계 의료 전문 답변)
- **AI 심의**: 닥톡 AI 의료광고 자동심의 (KMA 공식 파트너, 2~4개월 → 24시간)
- **광고 전략**: 닥톡 Ads DSP/DMP ($8B 헬스케어 광고 시장, Full-Cycle Performance Tracking)
- **재무 상태**: Default Alive, 35개월 런웨이 확보

자세한 내용은 [`business/docfriends-business-overview.md`](business/docfriends-business-overview.md) 참고.
