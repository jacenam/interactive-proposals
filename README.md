# 온라인 사업계획서

## 프로젝트 구조

```
온라인 사업계획서/
├── docs/                # 사업계획서 문서
├── resources/
│   ├── images/          # 이미지 및 로고
│   └── references/      # 참고 자료
├── financials/          # 재무 계획 및 예산
├── research/            # 시장 조사 자료
├── marketing/           # 마케팅 전략 자료
└── presentations/       # 발표 자료
```

## 폴더 설명

| 폴더 | 용도 |
|------|------|
| `docs/` | 사업계획서 본문, 요약서 등 |
| `resources/images/` | UI 목업, 로고, 스크린샷 |
| `resources/references/` | 외부 참고 문서 |
| `financials/` | 수익 모델, 예산, 투자 계획 |
| `research/` | 경쟁사 분석, 시장 조사 |
| `marketing/` | 마케팅 전략, 고객 분석 |
| `presentations/` | 투자자/발표용 자료 |

## 시작하기

1. `docs/` 폴더에 사업계획서 초안 작성
2. `research/` 폴더에 시장 조사 자료 수집
3. `financials/` 폴더에 재무 계획 수립
